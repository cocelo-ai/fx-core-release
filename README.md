# fx-core-release

Release package for **ARM64/AMD64 Ubuntu (py3.10/py3.11) support**.

## Download
```bash
git clone -b f<x>-core_<ver>_arm64 --single-branch https://github.com/cocelo-ai/fx-core-release.git
```
## Install

Download the `.deb` file from the **Assets** section, then install it with:

```bash
sudo apt install ./f<x>-core_<lib_ver>_arm64.deb
```
